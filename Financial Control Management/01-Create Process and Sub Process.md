# Create Process and Sub Process

## Prerequisites
Get access to IBM OpenPages by opening this [page](https://github.com/Client-Engineering-Indonesia/workshop-openpages/blob/main/Financial%20Control%20Management/00-Get%20your%20credentials) 

## Create Process

1. Click hamburger icon in top-left side -> click Organization -> click Process

<img width="259" height="387" alt="image" src="https://github.com/user-attachments/assets/8a88401d-79ee-4c44-801c-c21869497877" />

2. You will be redirected to Process page

<img width="1871" height="815" alt="image" src="https://github.com/user-attachments/assets/60042808-bf2e-4588-8104-110353c2402e" />

3. Click New button and fill the description as below

- Description:
```Purchasing and Payables```

- Process Owner:
```Search your username```

4. Scroll down until you find Primary Business Entity section -> click Select Primary Business Entity button -> select Finance

<img width="1720" height="673" alt="image" src="https://github.com/user-attachments/assets/fe09c092-2be8-4485-8c77-e4f8a823180a" />

5. Ensure you have same view as image below -> click Save button

<img width="1887" height="834" alt="image" src="https://github.com/user-attachments/assets/32b32782-2672-4457-b0e7-edf0e3ef47e3" />
