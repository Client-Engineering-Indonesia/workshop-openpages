# Perform Process Assessment

## Create Questionnaire Assessment

1. Go to your Process page -> scroll down to Process Questionnaire Assessment -> click Add New Questionnaire Assessment

<img width="1390" height="252" alt="image" src="https://github.com/user-attachments/assets/e467a77a-f3c1-494c-9431-83e5379b9b21" />

2. Set Assignee to your username -> click Select Questionnaire Template -> type ```Section 302 Template``` in Name search field -> select it and click Done

<img width="1715" height="666" alt="image" src="https://github.com/user-attachments/assets/0af312f9-b4e9-4724-94b4-4ffeadd40391" />

3. Click Save button

<img width="1883" height="888" alt="image" src="https://github.com/user-attachments/assets/7054afaa-61d5-4bbe-8bff-e24ddf2dd35c" />

## Review Questionnaire Assessment Workflow

1. Click gear icon -> select Solution Configuration -> select Workflows

<img width="266" height="265" alt="image" src="https://github.com/user-attachments/assets/bacad4b2-1cdb-440d-83d3-eea43fa1558d" />

2. Type "questionnaire" in search field -> click Questionnaire Assessment - POC

<img width="1902" height="415" alt="image" src="https://github.com/user-attachments/assets/322f8fef-11b4-49d6-9fe5-f37cda01c6e5" />

3. Here you can see workflow to complete questionnaire assessment

<img width="1895" height="848" alt="image" src="https://github.com/user-attachments/assets/04c8d0ac-23ea-47d8-825c-81a892c0007e" />

4. Click Complete Questionnaire node -> scroll down to Assignees and Subscribers section -> for the hands-on purposes assignee is set to your username through entire nodes

<img width="588" height="224" alt="image" src="https://github.com/user-attachments/assets/32709426-b59f-496e-ae42-e7442b73ff68" />


## Complete Questionnaire Assessment

1. Go back to your previous Questionnaire Assessment page -> click Action button -> select Start Questionnaire Assessment - POC

<img width="1900" height="726" alt="image" src="https://github.com/user-attachments/assets/c6d1148a-ad0c-462c-b0b4-7dd12fbbad93" />

2. Scroll down to Lifecycle Management section -> set Lifecycle to "2 Stage Lifecycle (no review)"

<img width="1383" height="323" alt="image" src="https://github.com/user-attachments/assets/e9545ae4-6eca-41fa-8bec-9fb15b77673e" />

3. Go to Questionnaire tab -> select Yes for question number 1 -> add "dummy notes" in comment section -> submit it

<img width="1892" height="694" alt="image" src="https://github.com/user-attachments/assets/c45c8850-e422-44bd-a780-428b7ea5ea84" />

4. For the remaining question you can answer with No. And for the statement you can set your name and add "dummy notes" in comment field
